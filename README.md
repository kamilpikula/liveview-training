# LiveView Studio

## Installation

1. Set up the project:

    ```sh
    mix setup
    ```

2. Fire up the Phoenix endpoint:

    ```sh
    mix phx.server
    ```

3. Visit [`localhost:4000`](http://localhost:4000) from your browser.

## Endpoints

Path  | Description
------------- | -------------
[`localhost:4000/light`](http://localhost:4000/light)  | The light that is controlled by buttons
